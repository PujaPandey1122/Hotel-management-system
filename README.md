🏨 HOTEL MANAGEMENT SYSTEM  

This project is a simple hotel management system developed using c++.
It demonstrates the use of object-oriented programming concepts such as classes ,objects,and inheritance .The system is designed to manage hotel operations including customer information ,room details and booking management.

The main purpose of this project is to understand how programming concepts can be used to build a structured application .It helps in learning program organizaton ,problem solving and teamwork while developing a practical software project.


✅FEATURES



-> Customer Registration 

->Room Booking 

->Room Cancellation

->Display Customer details


✅TECHNOLOGIES USED


->Language:- C++

->OOP(classes,inheritance,polymorphism,abstraction)
